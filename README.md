# Mystery-Temple-Run
MYSTERY TEMPLE RUN IS AN ENDLESS RUNNER GAME SET ON A SIMPLE PATH WHERE THE  PLAYER COLLECTS COINS AND TREASURES WHILE AVOIDING ENEMIES AND OBSTACLES.  ALONG THE PATH, THE PLAYER WILL ENCOUNTER MYSTERY TEMPLES. ENTERING A TEMPLE  TRANSPORTS THE PLAYER TO A NEW WORLD, EACH WITH ITS OWN THEME, ENVIRONMENT,  AND DIFFICULTY.
